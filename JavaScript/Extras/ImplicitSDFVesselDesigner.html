Implicit / SDF Vessel Designer

A first sandbox for vessel modeling from a scalar field F(x,y,z). The preview extracts the isosurface F=0 with marching tetrahedra and turns it into an open-top printable shell.
Choose a working family. It will load compatible body/detail sub-presets and a stable starting configuration.

Best results come from body equations that behave roughly like signed distance fields. The shell uses body + wall for the inner cavity, so very large non-SDF distortions can produce uneven wall thickness.

Equation helpers
  • Variables: x, y, z, r, R, theta, phi
  • Controls: radius, height, wall, baseThickness, openTopZ, detailAmp, smoothK, shapeExp
  • The shell now uses an approximate distance normalization F/|∇F| to keep walls more consistent on non-SDF equations.
  • Math: sin, cos, tan, abs, min, max, pow, sqrt, exp, log, atan2, clamp, mix
  • SDF-ish helpers: sphere(rad, ox, oy, oz), cappedCylinder(rad, halfH, ox, oy, oz), torus(major, minor, oz), box(bx, by, bz), superEllipsoid(rx, ry, rz, e)
  • Combiners: smin(a,b,k), smax(a,b,k), min(a,b), max(a,b)
Starter ideas
  • Pick a Global Preset first. It loads a working parameter set and compatible sub-presets.
  • Then swap the Body preset or Detail preset to explore nearby variations without breaking the shell.
  • Round bowl body: sphere(radius)
  • Blend body: smin(cappedCylinder(radius*0.72,height/2), sphere(radius*1.03,0,0,-height*0.02), smoothK)
  • Diagonal swirl detail: sin(7*theta + 0.8*z) * pow(max(0, 1 - abs(z)/(height*0.58)), 2)
